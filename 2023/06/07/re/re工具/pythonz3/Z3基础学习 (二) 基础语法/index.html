Z3基础学习 (二) 基础语法

一.设置变量

Int - 整数型

  • | 运算需要初始化为Int变量

Real - 实数型

BitVec - 向量（位运算）

  • 只有 BitVec 变量可以进行异或
  • BitVec 变量值之间可进行>或<或=或>=或<=的比较
  • BitVecVal 值之间不能进行>或<比较,只能转换成 python 认识的类型才可以比较

变量设置的类型可能会影响到最后求解的结果。可以先 check 一下看看有没有解，然后再判断是否需要切换变量的类型。
